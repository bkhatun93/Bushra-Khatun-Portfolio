Quick publish on GitHub Pages:
1. Sign in to GitHub.
2. Create a new public repository, for example: portfolio-website
3. Upload both files.
4. Open Settings > Pages.
5. Under Build and deployment, choose Deploy from a branch.
6. Select the main branch and the /root folder.
7. Save. Your site will go live in a minute or two.

After publishing:
- Copy your live URL
- Paste it into your report and website placeholder link
